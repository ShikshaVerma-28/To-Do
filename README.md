# ✅ To-Do List Application

A simple, efficient, and user-friendly To-Do list application to help you manage your daily tasks and boost productivity.

![Project Preview](Screenshot%202026-06-14%20154137.png)

## ✨ Features
* **Task Management:** Effortlessly add, edit, and delete your daily tasks.
* **Minimalist Design:** A clean interface that helps you focus on what matters most.
* **Responsive Layout:** Designed to work smoothly on all your devices.

## 🛠️ Tech Stack
* **Markup:** HTML5
* **Styling:** CSS3
* **Logic:** JavaScript
